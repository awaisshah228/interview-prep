## Webhooks & Real-Time Integration

### Webhook Patterns
| Pattern | Description |
|---|---|
| **Inbound Webhooks** | Receive events from Stripe, GitHub, Helius, etc. |
| **Outbound Webhooks** | Publish events to external consumers |
| **Signature Verification** | HMAC-based payload validation (Stripe signatures, GitHub secrets) |
| **Retry & DLQ** | Exponential backoff, dead-letter queues for failed deliveries |
| **Idempotency** | Deduplication using event IDs to prevent double processing |
